# Security Policy

## Scope

This repository hosts a public static website (HTML, CSS, JavaScript). The primary
security concerns are XSS vulnerabilities, injected content, and credential exposure.

## Reporting a Vulnerability

If you find a security vulnerability in the website code or CI/CD configuration,
please **do not open a public GitHub Issue**.

Instead, report it privately via email: <gamaware@gmail.com>

Include:

- The affected file or component
- A description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact

## Response

Reports will be reviewed and addressed promptly.
